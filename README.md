# 🎓 Student Registration System

A responsive **Student Registration System** built using **HTML, CSS, and JavaScript (DOM Manipulation)**.  
This project allows users to **add, edit, and delete student records**, with data stored in **localStorage** so it persists even after refreshing the page.

---

## 🔗 GitHub Repository

👉 https://github.com/roshnikri03/Student-Registration-System  

---

## 🚀 Features

- ✅ Add new student records  
- ✏️ Edit existing student details  
- ❌ Delete student records  
- 💾 Data stored in localStorage (no data loss on refresh)  
- 🔍 Input validation:
  - Name → only characters  
  - Student ID → only numbers  
  - Contact Number → minimum 10 digits  
  - Email → valid email format  
- 📱 Fully responsive (Mobile, Tablet, Desktop)  
- 📜 Dynamic scrollable student list  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (DOM Manipulation)
